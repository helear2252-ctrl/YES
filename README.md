# Machine Learning Portfolio

## 專案介紹

這是一個以「機器學習十大演算法」為主題的資料分析作品集網站，展示演算法知識、互動式線性迴歸、Outlier 分析、Raw Data 表格、CSV 匯出與 CRISP-DM 分析流程。

## 使用技術

* HTML5
* CSS3
* Vanilla JavaScript
* Chart.js
* GitHub Pages
* 未來可串接 Streamlit

## 專案結構

```text
Machine-Learning-Portfolio/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
```

## 核心功能

1. 12 個機器學習主題切換
2. 透明玻璃特效內容卡片
3. 互動式 Linear Regression Demo
4. Slider 即時控制資料變化
5. Outlier Detection
6. MAE / MSE / RMSE / R² 指標
7. Top 10 Outlier Observations
8. Raw Data Table
9. CSV Export
10. CRISP-DM Process Walkthrough

## 部署方式

本專案為純前端網站，可直接部署到 GitHub Pages。
